# WideScan README

## 概要

**WideScan** は、周囲のモンスター（またはエンティティ）をスキャンし、
**距離・方向・名前をリアルタイム表示する軽量アドオン**です。

いわゆる「簡易GPS＋広域スキャン」を目的としています。

---

## 主な機能

* 周囲のエンティティを自動スキャン
* 自分の座標を取得して表示
* 各モブとの距離を計算
* 相対方向（N / NE / E など）を表示
* 距離順でソート
* 最大20件まで一覧表示
* 軽量動作（スキャン間隔制御あり）

---

## 表示例

```
===== WideScan GPS =====
Self Pos: [123.45, 67.89]
----------------------------------
Goblin            12.3   NE
Orc               18.7   S
Bat               25.1   W
```

---

## 設定

### スキャン距離

```lua
local MAX_DISTANCE = 50
```

→ 表示する最大距離（単位：ゲーム内座標）

---

### スキャン間隔

```lua
local SCAN_INTERVAL = 0.1
```

→ 再スキャンまでの時間（秒）

* 小さいほどリアルタイム
* 大きいほど軽量

---

## 仕組み

### ① 自分の座標取得

送信パケット `0x015` から取得：

* X / Y / Z
* 向き（rot）

---

### ② エンティティスキャン

```lua
GetEntity(i)
```

で全インデックスを走査し：

* 名前あり
* HP > 0
* SpawnFlagsチェック（アクティブ対象）

のみ抽出

---

### ③ 距離計算

```lua
sqrt(dx^2 + dy^2)
```

※ XY平面のみ（高さ無視）

---

### ④ 方向計算

```lua
atan2(dx, dy)
```

→ 角度 → 8方向へ変換：

* N / NE / E / SE / S / SW / W / NW

---

### ⑤ ソート

```lua
table.sort(mobs, function(a, b)
    return a.d < b.d
end)
```

→ 近い順で表示

---

## 表示仕様

* フォント：Consolas
* サイズ：12px
* 左上に固定表示
* 最大20件表示

---

## 注意点

### ■ 自分座標が取れない場合

```
Self Pos: waiting packet...
```

→ 移動するまで表示されない場合あり

---

### ■ 高さ（Z）は無視

距離は**平面距離のみ**

---

### ■ 全エンティティ走査

```lua
for i = 0, 2303
```

→ 負荷は軽いが完全無料ではない
